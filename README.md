# Dash-Dough Bird [![Hackatime](https://img.shields.io/badge/Hackatime-11h_10m-ff477e?logo=wakatime&logoColor=white)](https://github.com/DilsanBasnet)

## DEMO:  [DASH DOUGH BIRD](https://dilsanbasnet.itch.io/dashdoughbird)

*This is a Dash-Dough Bird a 2D arcade project build entirely from scratch. It is a replica of the classic Flappy Bird, but I have added a bunch of cool custom features of make it way more exciting and challenging..*

## Controls: 

- Jump: Spacebar, W & Up arrow key.

- Play, Exit Button: Click with mouse.

## Game Scenes: 
- MainMenu Scene - GameNameImage, Play & Exit Button, Soft Music Starts
- MainGame Scene - DashDoughGameScene & GameOverScene , Play Button, Exit Button, Sounds Effects 

## Build : 
- Unity as GE
- C# for Scripting
- Game Title Image - Pixlr
- Sprites: Sourced via Pinterest
- Music & Sound Effects: Sourced via OpenGameArt
  
## Features: 
- Game gets faster and harder every time player cross the pipes.

- The pipes automatically spawn faster as the game speed up also keep the distance between pipes clone. 

- Collectable Item (Diamonds) collect diamonds floating while player dodge pipes obstacles. 

- Added soft background music and sound effect when player grab diamonds and sound effect when player crash with pipes. 

- Working Main Menu & Game Over Scenes with Play & Exit Buttons.
